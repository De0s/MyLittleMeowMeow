# Команды GIT
## Создание репозитория
```
git init
```
> создает репозиторий в выбранной папке
```
git add .
```
> добавляет файлы в репозиторий
```
git status
```
> показывает статус репозитория
```
git commit -m "commit"
```
> сохраняет изменения
```
ls -la
```
